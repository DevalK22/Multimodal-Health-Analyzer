How to use ?

This daigonsis website were you can get results by uploading X-ray, MRI, textual based data which can save so much time for doctor.

What you'll get

  1. Click on "Start" Latest

    Search the "Start" button below or at the nav bar.

  2. Choose the daigonsis

    Choose the disease between Brain Tumor, Alzheimer, Diabetes, Pneumonia, Heart Disease, Symptoms Based.

  3. Upload image

    You can upload "jpg", "jpeg", "jfif", "pjpeg", "pjp", "png", "webp" format images and you can use drag and drop too.

  4. Enter "Daigonsis"

    Click Daigonsis and wait for result to preview. The preview page will show daigonsis and its types with key information.

Project Information

In this project, we developed a Multimodal health analyzer with a primary focus on analyzing X-ray images and providing disease predictions based on both visual data and manually input symptoms. The project utilized machine learning algorithms to train the system on X-ray datasets, enabling accurate detection of specific diseases. Additionally, a user-friendly interface was developed using Flask for easy interaction. As we move into the next phase of the project, MRI and CT scan analysis will be integrated, further enhancing the system's diagnostic capabilities.
